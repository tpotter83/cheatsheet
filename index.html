Terms to Know

Term Meaning
Typography The art of arranging text on a page
Serif Font Fonts that contain extra details on the ends of each letter
Sans-Serif Font Fonts that do not contain details on the ends of each letter
Fallback Font A font that will be used if the font(s) declared before are not available on the user’s computer
Font Stack A series of declared fonts to be used in the order specified
Web Safe Fonts Fonts that will appear the same across all browsers and operating systems

Typography Properties

Property Accepted Values and/or Notes
Font-weight Bold, normal, lighter, bolder, numerical value between 1 – 1000
*A numerical value of 400 is equal to normal and 700 is equal to bold*
Font-style Normal (default), italic, oblique
Text-transform Length values such as px or em
Letter-spacing Length values such as px or em
Word-spacing Length values in units such as px or em
Font-family Any defined or linked font

Linking & Using Fonts

Code How to Use
HTML <link href=”url” rel=”stylesheet”>
CSS @font-face { Font-family: ‘define name’; Src: url(‘file path or url’) format(‘file type’); }
CSS Copy url of font and paste into browser, copy the /* latin */ @font-face code and paste into CSS file
